# Hospital Management System 

A comprehensive desktop application to manage hospital operations efficiently, built using Java Swing for the GUI, JDBC for database connectivity, and MySQL as the backend. Designed to handle patients, staff, appointments, and billing with ease.

# Features 

Patient Management – Add, update, view, and delete patient records.

Doctor & Staff Management – Maintain detailed staff profiles and schedules.

Appointment Scheduling – Book and track patient appointments effortlessly.

Billing & Payments – Automatically calculate and record treatment charges.

Search & Filter – Quickly find patients, doctors, or staff.

Database Integration – Persistent storage using MySQL.

User-Friendly GUI – Sleek Swing-based interface for easy navigation.

# Technologies Used 

Java – Core application logic

Swing – Desktop GUI development

JDBC – Database connectivity

MySQL – Relational database management
